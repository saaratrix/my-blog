I wanted to learn Unreal Engine 4 better and decided to watch all Unreal Engine 4 live training youtube videos by Epic Games. The videos taught me about a wide variety of features that the engine has.

Introduction

I've been wanting to learn Unreal Engine 4 for quite some time and the first time I briefly used the engine was in 2015 with version 4.7. I used 4.7 for maybe 60 hours in total. I also used UE3 or UDK back in 2012 but I never got very far with unrealscript. But in May 2017 I decided to give Unreal Engine another go and came across a live training stream Epic has been running about once a week since 2014. The videos I watched can be found at Unreal Engine's youtube channel in this youtube playlist. I decided to try and watch all of the videos to learn about the engine itself and other 3D concepts.

I set out a goal to watch 1 video a day and started on the 8th of May and finished on September 16th. Watching the videos with 1.25x speed saved me about 40 hours in total :), so that's a handy feature! Some videos were more interesting than others which made me not always watch the video with 100% focus. I particularly liked watching the Alan Willard videos that had a lot of eye candy effects such as disintegration. Videos about VR became a history timeline about Oculus Rift, Samsung Gear VR and eventually HTC Vive.

I took some notes from almost every video that was mainly for myself in case I need to search on an issue to find a possible video about the issue. However I figured the notes could serve as an overview of what the video is about. The people uploading the youtube videos sometimes did really good job of adding timestamps of each topic in the video. Some of the notes can be quite cryptic and if they are hard to understand I apologise in advance. In this blog post there is also a C# console application snippet to parse the CSV note file into HTML output because editing 90 rows by hand would have been quite time consuming!

C# CSV Parser source code

This is the source code for the C# console application to automatically generate the HTML code for this blog post. It parses a csv exported by google documents using the TextFieldParser from Microsoft.VisualBasic.dll. 

        
namespace CSVParser
{
    class Program
    {
        static void Main(string[] args)
        {
            List<string[]> rows = GetRows("uelivetraining.tsv");
            string output = CreateOutput(rows);
            string outputToC = CreateOutputToC(rows);

            File.WriteAllText("output.txt", output);
            File.WriteAllText("output.toc.txt", outputToC);
        }

        static List<string[]> GetRows(string a_filename)
        {
            bool firstParse = false;

            List<string[]> rows = new List<string[]>();

            using (TextFieldParser parser = new TextFieldParser(a_filename))
            {
                parser.TextFieldType = FieldType.Delimited;
                parser.SetDelimiters("\t");

                while (!parser.EndOfData)
                {
                    //Process row
                    string[] fields = parser.ReadFields();

                    if (!firstParse)
                    {
                        firstParse = true;
                        continue;
                    }

                    rows.Add(fields);
                }
            }

            return rows;
        }

        static string CreateOutput(List<string[]> a_rows)
        {
            string output = "";
            StringBuilder outputBuilder = new StringBuilder();

            outputBuilder.AppendFormat("<dl>" + Environment.NewLine);

            foreach (string[] row in a_rows)
            {
                string title = row[0];
                string description = row[1];
                string videolink = row[2];

                string titleUnderscored = title.Replace(' ', '_');

                outputBuilder.AppendFormat(
                    "\t<dt>" + Environment.NewLine +
                        "\t\t<a name=\"{0}\">{1}</a>" + Environment.NewLine +
                    "\t</dt>" + Environment.NewLine +
                    "\t<dd>" + Environment.NewLine +
                        "\t\t{2}" + Environment.NewLine +
                        "\t\t<br>" + Environment.NewLine +
                        "\t\tVideo: <a href='{3}'>{3}</a>" + Environment.NewLine +
                    "\t</dd>" + Environment.NewLine
                    , titleUnderscored, title, description, videolink);
            }

            outputBuilder.AppendFormat("</dl>" + Environment.NewLine);

            output = outputBuilder.ToString();

            return output;
        }

        static string CreateOutputToC(List<string[]> a_rows)
        {
            string output = "";
            StringBuilder outputBuilder = new StringBuilder();

            outputBuilder.AppendFormat("<ol>" + Environment.NewLine);

            foreach (string[] row in a_rows)
            {
                string title = row[0];
                string titleUnderscored = title.Replace(' ', '_');

                // For the ordered list inside the <details> tag
                outputBuilder.AppendFormat(
                    "\t<li>" + Environment.NewLine +
                        "\t\t<a href=\"#{0}\">{1}</a>" + Environment.NewLine +
                    "\t</li>" + Environment.NewLine, titleUnderscored, title);
            }

            outputBuilder.AppendFormat("</ol>" + Environment.NewLine);

            output = outputBuilder.ToString();

            return output;
        }
    }
}
